# Item Nameplate

[English](Home)

Item Nameplate 是一个 Minecraft 1.20.1 Forge 客户端模组。它会在容器和快捷栏的物品图标上显示可配置的紧凑名称牌。

名称牌按物品实际字体宽度截取，适配中英文混排；文字覆盖物品材质，但会被原版提示框自然遮挡。

## 支持环境

| 项目 | 版本 |
| --- | --- |
| Minecraft | 1.20.1 |
| Forge | 47.4.20 或兼容版本 |
| Java | 17 |

## 主要功能

- 通过 JSON 为单个物品、物品标签或物品类配置名称牌。
- 从物品名称、NBT 或提示框指定行读取文字。
- 支持删除文字、替换文字、翻译键本地化、拆分和拼接。
- 内置中英文兼容规则：附魔书、刷怪蛋、药水、药水箭、锻造升级模板和盔甲纹饰模板。
- 在背包、容器和快捷栏中显示名称牌。
