# Item Nameplate

[中文](Home-ZH)

Item Nameplate is a Minecraft 1.20.1 Forge client mod that renders compact, configurable labels over item icons in containers and the hotbar.

Labels are clipped by actual font width, support mixed Chinese and English text, render above item textures, and are naturally covered only by the part of a tooltip that overlaps them.

## Requirements

| Component | Version |
| --- | --- |
| Minecraft | 1.20.1 |
| Forge | 47.4.20 or compatible |
| Java | 17 |

## Features

- Configure labels for an item ID, item tag, or item class with JSON rules.
- Read text from an item name, NBT string, or a chosen tooltip line.
- Remove and replace text, localize translation keys, and split or join values.
- Includes Chinese and English rules for enchanted books, spawn eggs, potions, tipped arrows, smithing upgrade templates, and armor trim templates.
- Renders in inventories, containers, and the hotbar.
